Contribuez au développement de METEOR


METEOR est un logiciel open source, vous pouvez donc participer activement à son développement.

  • Comment signaler ou corriger un bug ?
    1. Aller sur la page principale du dépôt github contenant le bug.
    2. Sous le nom du dépôt, cliquer sur "Issues"
    3. Cliquer sur le bouton vert intitulé "New Issue"
    4. Entrez le titre et une description du bug
    5. Lorsque vous avez fini, cliquez sur "Submit new issue"

  • Comment proposer une amélioration ?
    1. Commencez par cloner le dépôt qui vous intéresse
    2. Créez localement une nouvelle branche et faites vos modifications dedans
    3. Lorsque vous êtes satisfait de vos modifications, committez et pushez les sur votre dépôt cloné.
    4. Naviguez sur le dépôt original
    5. Dans l'onglet de base (Code), à droite des branches, cliquez sur "New pull request"
    6. Sur la page "Compare", cliquez sur "compare across forks"
    7. La "base fork" doit être le dépôt dans lequel vous souhaitez introduire vos modifications. Utilisez le menu déroulant "branch" pour choisir la branche dans laquelle vous souhaitez fusionner vos modifications
    8. La "head fork" doit être le dépôt cloné dans lequel vous avez fait vos modifications. Utilisez le menu déroulant "branch" pour choisir la branche dans laquelle vous avez réalisé vos modifications
    9. Entrez le titre et la description de vos modifications
    10. Cliquez sur "Create pull request"

  • Comment proposer une intégration avec un outil ou logiciel existant ?
    1. Vous pouvez suivre les mêmes indications que ci dessus pour proposer une amélioration
    2. Dans le cas d'un besoin spécifique, vous pouvez nous contacter ici
